# Huffman Tree Practice

本实验用于复习 408 数据结构中的树、二叉树、森林、哈夫曼树、队列和栈。

## 涉及知识点

- 多叉树的孩子兄弟表示法
- 多叉树转二叉树
- 二叉树先序遍历
- 二叉树层序遍历
- 循环队列
- 栈与非递归遍历
- 优先队列
- 哈夫曼树构造
- 哈夫曼编码

## 核心思路

1. 使用 `firstkid` 和 `nextbrother` 表示多叉树。
2. 将多叉树转换成二叉树。
3. 拆分二叉树中的所有节点。
4. 使用小根堆每次取出两个权值最小的节点。
5. 合并两个节点，生成新的父节点。
6. 重复合并，直到只剩一棵树。
7. 从根节点出发，左分支记为 `0`，右分支记为 `1`，生成哈夫曼编码。
